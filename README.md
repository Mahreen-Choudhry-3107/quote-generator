
# Quote Generator

A modern Quote Generator built with React, TypeScript, and Vite. This application displays inspirational quotes and allows users to generate new quotes instantly.

## Features

* Generate random quotes
* Clean and responsive UI
* Fast performance with Vite
* Built using React and TypeScript
* Easy to customize and extend

## Tech Stack

* React
* TypeScript
* Vite
* CSS
